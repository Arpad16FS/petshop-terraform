
# petshop-terraform

Componente de IaC para el despliegue de una aplicación con interfaz web haciendo uso de los servicios de AWS para la prestación de servicio de agendamiento de baño, estilista, y demás servicios para mascotas a domicilio.

## Instalación

- 1️⃣ Clona el repositorio.
- 2️⃣ Configura tus credenciales de segurridad o deja las por defecto.
- 3️⃣ Ejecuta el manifiesto dentro de tu entorno de AWS.
- 4️⃣ Modifica el frontend a las necesidades de tu negocio.

## Frontend y aplicación en Node.js

En esta versión se crea a través de un user_data configurado de forma rígida en este mismo código fuente. Se planea hacer uso de un dockerfile o imagen preconfigurada de AWS para un mejor mantenimiento de esta parte del código.
